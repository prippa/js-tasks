Задание 1:

Написать функцию, которая принимает на вход строку и возвращает самую длинную подстроку с неповторяющимися символами.

Например: 
abcabcbb -> abc
bbbbb -> b
pqqkeq -> qke

Задание 2:

Написать функцию, которая принимает положительное число и возвращает число из цифр в обратной последовательности. Задача должна быть выполнена без преобразования чисел в строку.

Например: 
15 -> 51
123 -> 321
3012 -> 2103
20 -> 2

Задание 3:

1) №
2) Name
3) E-mail
4) Age

и какими-то произвольными данными. Нужно сделать таблицу сортируемой с использованием только VueJS, без плагинов и других библиотек. Пример: https://mdbootstrap.com/docs/jquery/tables/sort/
